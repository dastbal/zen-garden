Style Guide 1.0

Purpose/Theme

The forest and mountains as described by John Muir “are a place of healing.” With its greens, and calming tans. It is a theme that will instill a Zen while providing a contrast that makes information gathering pleasing to the eye. The theme will feature tree silhouettes and contrast type overlaying type boxes of green and tan. At the bottom of the page will include trees for the type and headings to rise above as if a moon ascending above the forest below.
